# Inclusive Brand Guide Template

This project is a template for creating an inclusive brand guide, built with [Astro](https://astro.build).

## Prerequisites

- Node.js (v18.14.1 or higher recommended)
- npm (comes with Node.js)

## Installation

1.  Open your terminal.
2.  Navigate to the project directory.
3.  Install the dependencies:

    ```bash
    npm install
    ```

## Running Locally

To start the development server and view the site locally:

1.  Run the development command:

    ```bash
    npm run dev
    ```

2.  Open your browser and navigate to `http://localhost:4321`.

## Building for Production

To build the site for deployment:

```bash
npm run build
```

This will generate a `dist/` directory with your static site.
